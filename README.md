# 🚀 2025双11电商消费趋势深度复盘与管理系统

这是一个基于 Python 生态构建的端到端电商数据处理与分析案例。项目模拟了 2025 年双11期间的真实业务场景，通过构建自动化数据流，实现了从数据建模、业务逻辑注入到 Web 交互式分析的全链路覆盖。

## 🌟 项目亮点
- **消费分级洞察**：重点分析一线城市与下沉市场（三线及以下）在价格区间、折扣敏感度上的行为差异。
- **全链路技术栈**：涵盖了数据库建模、数据清洗、可视化分析及 Web 应用开发。
- **交互式管理后台**：基于 Streamlit 构建，支持实时数据录入、商品信息快捷编辑及动态业务看板。

## 🛠️ 技术架构
- **核心语言**: Python 3.13
- **数据库层**: SQLite + SQLAlchemy (实现 ORM 对象关系映射)
- **数据分析**: Pandas (数据透视与清洗)
- **数据可视化**: Matplotlib + Seaborn
- **Web 框架**: Streamlit (构建交互式决策支持系统)
- **版本控制**: Git (规范化的 .gitignore 配置)

## 📂 项目结构说明
- `app.py`: Web 管理系统入口，包含实时看板和数据录入功能。
- `data_generator.py`: 数据库结构定义脚本，负责初始化表结构。
- `fill_data.py`: 业务逻辑注入脚本，利用 Faker 模拟万级高仿真业务数据。
- `analysis.py`: 核心分析逻辑，计算 GMV、客单价及折扣率。
- `visualize.py`: 静态图表生成脚本，用于复盘报告的快速出图。
- `double11_2025.db`: 本地 SQLite 数据库文件（已在 Git 中忽略）。
